# Technical Interview

This repository contains the base code for technical interviews at Mirico.

## Installing

The reccomended way to use this repository is through GitHub Codespaces or Dev Containers. To get started, simply open this repository in GitHub Codespaces or clone it locally and open it in VS Code with the Dev Containers extension installed.

## Challenges

The take-home assignment is located in the `challenges` folder. Read the README.md file in that folder for more information about the take-home assignment.
